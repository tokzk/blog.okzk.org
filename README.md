
## Create draft

    $ jekyll draft "How to create draft with such title"

## Move to post

    $ jekyll publish _draft/how-to-create-draft-with-such-title.md --date 2016-01-24

## Build site

    $ jekyll build

## Launch Server

    $ jekyll serve

## Lunch Server with draft

    $ jekyll serve --drafts

## Build production site and deploy

    $ ./bin/deploy

## deploy

    $ jekyll deploy
